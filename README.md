# Постановка задачи

Дано описание синтаксической структуры языка с помощью контекстно-свободной (КС) грамматики. Необходимо:
1. **Провести лексический анализ** для выделения имеющихся в языке лексем. Лексический анализатор должен быть устойчив к добавлению произвольного числа пробельных символов между лексемами.
2. **Выполнить грамматический разбор** с применением метода рекурсивного спуска. При необходимости грамматику необходимо модифицировать так, чтобы метод рекурсивного спуска стал применим.
3. **Провести семантические проверки** для заданного языка. Необходимо проверить корректность типов в выражениях и отловить ошибки, связанные с необъявленными или неинициализированными переменными.

Программа должна выводить пользователю осмысленные сообщения об ошибках разного уровня.
Вместе с кодом программы к ответу должны быть приложены корректные и некорректные примеры входных данных. Некорректные данные должны быть представлены для всех возможных видов ошибок (синтаксические, разного вида семантические).
Также программа реализует генерация внутреннего представления кода программы в формате ПОЛИЗа.

# Используемые обозначения

Фигурные скобки $\{a\}$ означают повторение строки произвольное количество раз. К примеру, выражение $a\{a\}$ допускает строки: $a$, $a, a$, $a, a, a$ и так далее.
**Жирным** шрифтом выделены ключевые слова языка программирования.
Круглые `()` и квадратные `[]` скобки — обычный элемент грамматики, если не указано обратное.
Через знак «равно» указаны шаблоны для токенов языка с помощью языка регулярных выражений.
Символ `&` зарезервирован для обозначения служебного токена конца программы, вводимого для удобства описания программы.

# Описание языка

Данный язык поддерживает целочисленный и булев типы. В языке нет логических операций. Поддерживается цикл `while` и условный оператор `if` с опциональной веткой `else`.

### Грамматика

```text
Program → VarDecls Body

VarDecls → {VarDecl ;}

VarDecl → Id : Type

Type → int
     | bool

Body → begin {Statement ;} end

Statement → Id = Comp
          | if Comp then Body
          | if Comp then Body else Body
          | while Comp Body
          | print(Expr)

Comp → ArithExpr == ArithExpr
     | ArithExpr != ArithExpr
     | ArithExpr > ArithExpr
     | ArithExpr < ArithExpr
     | BoolNum
     | ArithExpr

ArithExpr → ArithExpr + Term
          | ArithExpr − Term
          | Term

Term → Term * Factor
     | Term / Factor
     | Factor

Factor → Id
       | Num
       | (ArithExpr)

Id = [a-z][a-Za-Z0-9]*

Num = [0-9]+

BoolNum = True | False
```
